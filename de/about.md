# Über Board Game Clubs

Board Game Clubs ist ein kostenloses, von der Community gepflegtes Verzeichnis von Brettspielclubs. Das deutsche Verzeichnis findest du hier; britische Clubs sind unter [boardgameclubs.org]({{ "/" | absolute_url }}) gelistet. Wir verbinden Menschen mit lokalen Clubs, damit du leicht herausfinden kannst, wo und wann sich Gruppen in deiner Nähe treffen.

## So funktioniert es

- Stöbere auf der [Karte]({{ "/de/" | relative_url }}), um Clubs in deiner Nähe zu finden
- Filtere nach Wochentag oder suche nach Name und Ort
- Klicke auf einen Club für alle Details: Veranstaltungsort, Zeit, Kosten und Links
- Nutze „Meinen Standort verwenden", um Clubs nach Entfernung zu sortieren

## Von der Community gepflegt

Alle Clubdaten liegen als einzelne Dateien in unserem [GitHub-Repository](https://github.com/BoardGameClubs/BoardGameClubs_Web). Jede:r kann einen Pull Request einreichen, um einen neuen Club hinzuzufügen oder bestehende Informationen zu aktualisieren. Details findest du in unserem [Beitragsleitfaden]({{ "/de/contribute/" | relative_url }}).

## Keine Konten, datenschutzfreundlich

Board Game Clubs ist eine statische Website, gehostet auf GitHub Pages. Es gibt keine Benutzerkonten und keine Cookies. Für anonyme Besucherstatistiken nutzen wir [GoatCounter](https://www.goatcounter.com/) — es sammelt keine persönlichen Daten, verwendet keine Cookies und ist vollständig Open Source. Die interaktive Karte basiert auf OpenStreetMap, das ebenfalls kostenlos und Open Source ist.

## Mitmachen

Kennst du einen Brettspielclub, der noch nicht gelistet ist? [Füge ihn dem Verzeichnis hinzu]({{ "/de/contribute/" | relative_url }}), indem du einen Pull Request auf GitHub einreichst. Das dauert nur wenige Minuten.
